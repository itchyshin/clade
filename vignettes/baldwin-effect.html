The Baldwin Effect: within-lifetime learning accelerating evolution

Shinichi Nakagawa

Background

The Baldwin Effect (Baldwin 1896; Simpson 1953) is a neo-Darwinian mechanism by which phenotypic plasticity — the ability of an organism to adjust its behaviour within its lifetime — can accelerate genetic evolution without Lamarckian inheritance. The logic is as follows:

  1. In a novel environment, genetically fixed behaviours are suboptimal.
  2. Individuals that can learn (or copy successful neighbours) survive long enough to reproduce, even when their genome does not yet encode the optimal behaviour.
  3. Because learners reproduce more, genetic variants that predispose faster or deeper learning spread in the population.
  4. Over generations, the benefit of learning decreases as the genome gradually encodes the once-learned behaviour directly — a process called genetic assimilation (Waddington 1942).

clade provides two within-lifetime learning modules that can be used independently or together to study these dynamics:

  • Within-lifetime RL (rl_mode = "actor_critic"): REINFORCE with baseline updates the action-selection layer of an agent’s neural network each tick, driven by energy-gain reward.
  • Social learning (social_learning = TRUE): prestige-biased copying — each agent blends a fraction of the highest-energy neighbour’s output-layer weights into its own policy, following the strategy of Henrich & Gil-White (2001).

Minimal demonstration

The experiment below compares three conditions over 300 ticks on a 30×30 grid:

Condition rl_mode social_learning Expected outcome
Baseline "none" FALSE Slowest adaptation
RL only "actor_critic" FALSE Faster individual adaptation
RL + SL "actor_critic" TRUE Fastest cultural spread 
library(clade)
library(ggplot2)
library(patchwork)
base <- default_specs()
base$grid_rows     <- 30L
base$grid_cols     <- 30L
base$n_agents_init <- 80L
base$max_agents    <- 500L
base$max_ticks     <- 300L
base$random_seed   <- 42L
# Start in a low-resource environment to create selection pressure for learning
base$grass_rate    <- 0.03
base$grass_init_prob <- 0.3
# Condition 1: no within-lifetime learning
s_base <- base
s_base$rl_mode        <- "none"
s_base$social_learning <- FALSE
env_base <- run_alife(s_base)

# Condition 2: individual RL only
s_rl <- base
s_rl$rl_mode        <- "actor_critic"
s_rl$rl_update_freq <- 1L
s_rl$social_learning <- FALSE
env_rl <- run_alife(s_rl)

# Condition 3: RL + prestige-biased social learning (full Baldwin pathway)
s_both <- base
s_both$rl_mode             <- "actor_critic"
s_both$rl_update_freq      <- 1L
s_both$social_learning      <- TRUE
s_both$social_learning_freq <- 5L
s_both$social_learning_rate <- 0.1
env_both <- run_alife(s_both)

Comparing population trajectories

d_base <- get_run_data(env_base)$ticks
d_rl   <- get_run_data(env_rl)$ticks
d_both <- get_run_data(env_both)$ticks

# Combine into a single tidy data frame
d <- rbind(
  transform(d_base, condition = "Baseline"),
  transform(d_rl,   condition = "RL only"),
  transform(d_both, condition = "RL + SL")
)
d$condition <- factor(d$condition, levels = c("Baseline", "RL only", "RL + SL"))
p_agents <- ggplot(d, aes(t, n_agents, colour = condition)) +
  geom_line(linewidth = 0.8) +
  labs(x = "Tick", y = "Population size",
       colour = NULL, title = "Population size") +
  theme_minimal(base_size = 12)

p_energy <- ggplot(d, aes(t, mean_energy, colour = condition)) +
  geom_line(linewidth = 0.8) +
  labs(x = "Tick", y = "Mean energy",
       colour = NULL, title = "Mean energy per agent") +
  theme_minimal(base_size = 12)

p_agents / p_energy + plot_layout(guides = "collect") &
  theme(legend.position = "bottom")

The RL + SL condition typically reaches higher mean energy sooner than either the baseline or RL-only condition, reflecting the two-step Baldwin pathway: individual reinforcement discovers locally good policies, while social copying spreads successful policies faster than genetic transmission alone.

Tracing genetic assimilation

Genetic assimilation leaves a signature in the population’s genetic diversity: early in the run, diversity is maintained because the genome does not yet encode the optimal policy; later, as the learned behaviour becomes genetically fixed, diversity decreases as a small number of well-adapted genotypes dominate.

ggplot(d, aes(t, genetic_diversity, colour = condition)) +
  geom_line(linewidth = 0.8) +
  labs(x = "Tick", y = "Mean pairwise genome distance",
       colour = NULL,
       title = "Genetic diversity — signature of genetic assimilation") +
  theme_minimal(base_size = 12) +
  theme(legend.position = "bottom")

A steeper late-run decline in the RL + SL condition relative to the baseline is consistent with accelerated assimilation: the learning buffer reduces selection variance early, allowing more genetic variants to persist, but the benefit of learning narrows once the genome catches up, driving a sharper selective sweep.

Combining with epigenetics

The epigenetics = TRUE module adds a second non-genetic inheritance channel: methylation marks accumulated during an agent’s lifetime are transmitted to offspring with probability epigenetic_inheritance. This can accelerate the Baldwin pathway further because an offspring inherits not only its parent’s genome but also a prior on which synaptic weights to canalize.

s_epi <- s_both
s_epi$epigenetics            <- TRUE
s_epi$epigenetic_inheritance <- 0.5
s_epi$methylation_rate       <- 0.2
s_epi$demethylation_rate     <- 0.05
env_epi <- run_alife(s_epi)
d_epi <- get_run_data(env_epi)$ticks
d2 <- rbind(
  transform(d_both, condition = "RL + SL"),
  transform(d_epi,  condition = "RL + SL + Epigenetics")
)
ggplot(d2, aes(t, mean_prior_sigma, colour = condition)) +
  geom_line(linewidth = 0.8) +
  labs(x = "Tick", y = "Mean BNN prior sigma",
       colour = NULL,
       title = "BNN uncertainty — canalization via epigenetics") +
  theme_minimal(base_size = 12) +
  theme(legend.position = "bottom")

mean_prior_sigma (the mean standard deviation of BNN weight distributions) measures how certain agents are about their action-selection policy. Lower values indicate more canalized — and potentially more heritably fixed — neural circuits. The epigenetic condition should show faster sigma reduction because offspring inherit methylation marks that bias sigma contraction towards the parent’s effective policy.

Quantifying the effect

estimate_heritability() provides a rough proxy for the heritability of the learned component:

h2_base <- estimate_heritability(env_base, trait = "mean_energy")
h2_rl   <- estimate_heritability(env_rl,   trait = "mean_energy")
h2_both <- estimate_heritability(env_both, trait = "mean_energy")

cat(sprintf("Heritability proxy (lag-1 autocorrelation):\n"))
cat(sprintf("  Baseline:  %.3f\n", h2_base$h2))
cat(sprintf("  RL only:   %.3f\n", h2_rl$h2))
cat(sprintf("  RL + SL:   %.3f\n", h2_both$h2))

Higher autocorrelation in the RL + SL condition reflects a more stable (genetically assimilated) population trajectory: once the optimal policy is encoded in the genome, mean energy varies less from tick to tick.
