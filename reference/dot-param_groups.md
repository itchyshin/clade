# List the named parameter groups (internal — used by vignettes)

Returns the names of `.SPEC_GROUPS` in display order. Used by
[`vignette("parameter-reference")`](https://itchyshin.github.io/clade/articles/parameter-reference.md)
to iterate over groups programmatically.

## Usage

``` r
.param_groups()
```

## Value

Character vector of group names.
